看山是山，看水是水；看山不是山，看水不是水；看山还是山，看水还是水。

变的是自己。

对于音乐文件的管理，我现在的状态是：既没有高要求，又不愿太费劲。展开来说就是 128 Kbps 的 mp3 就能满足我，不需要 XXXX Kbps 的 Flac 了；对于特定的音乐文件，不需要完完整整的元数据，只需要包含正确的专辑封面、基本的标签即可（一度我甚至连这些也不在意，没有就留空呗）。我不愿意自己动手去处理它们，简言之就是懒惰战胜了洁癖。

所以我用自带的 Windows Media Player（以下简称 WMP），特别是当我发现它本身就支持联网添加、修正歌曲的标签时。在 Google 宣布停止在中国的音乐服务后，我没有办法使用谷歌音乐盒批量下载歌曲了，只好转到百度音乐。这时，歌曲的标签问题才体现出来：百度音乐提供的歌曲信息有问题。举例来说，一首本该叫做「Don't Think About Me」（虚构）的歌曲可能会变为「Don’T Think About Me」。

而 WMP 可以帮我纠正。

把歌曲传输到移动设备上也是很方便的，把歌曲拖动到右侧窗格，点击同步即可。

完成后在移动设备上的目录结构是：「ARTIST（父目录）-- ALBUM（子目录）-- TRACKS（音乐文件们）」

Enjoy!
