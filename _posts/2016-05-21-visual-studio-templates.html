Differences Between Project and Item Templates 介绍了「项目模板」和「项模板」，通过前者在新建项目时弹出，后者在项目中添加新项时弹出。

安装好的模板是以 .zip 的形式存放于特定目录下，具体见 How to: Locate and Organize Project and Item Templates。

  • VS安装目录\Common7\IDE\ItemTemplates\Language\
  • VS安装目录\Common7\IDE\ProjectTemplates\Language\

组成项目模板的内容：

  • .vstemplate 文件，包含了该模板的元数据
  • 项目文件，例如 .csproj
  • 代码文件、资源文件
  • 可选的图标

组成项模板的内容：

  • .vstemplate 文件，包含了该模板的元数据
  • 特定的要添加的文件
  • 可选的图标

目前我有个需求，某 SDK 提供了一套在 Visual Studio 2010 下的模板安装包，但是无法在更高版本的 Visual Studio 下安装；掌握上面的内容后，就可以手工完成：

  • 把低版本 VS 下的相应的模板文件 .zip 拷贝到高版本的 VS 对应的目录下
  • 关闭所有 Visual Studio 实例
  • 然后在开始菜单内，找到高版本的 VS Developer Command Prompt，运行：devenv /installvstemplates
  • 最后，打开高版本 VS 验证
