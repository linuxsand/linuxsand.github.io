华为 Android 手机获取 ROOT 权限小记

黄杰, 2011-09-13
root[a]linuxsand.info

ROOT 过 3 台华为的 Android 机型——C8500，C8500S，C8650。

为什么要 ROOT？因为这样就可以删除电信定制应用，我不喜欢直接刷第三方的 ROM，其往往会集成常用软件，这是我不需要的；我总是偏爱纯净系统，安装少量第三方软件满足硬需求即可。

获取 ROOT 权限是第一步，之后需要安装诸如 ROOT Explorer 这样的软件来删除定制应用。

C8500

我自己使用的就是这一机型，最简单的方法是使用 z4root。提权后安装 ROOT Explorer 删除定制应用，位置：/cust/chinatelcom/cn/app & /system/app

C8500S

无法使用 z4root，可行的办法是在 PC 上使用 SuperOneClick.exe 来获取 ROOT 权限。

前提是：

  • 在手机端：设置 - 应用程序 - 开发 - 勾选 USB 调试
  • 在 PC 上安装手机驱动程序
  • 运行 SuperOneClick.exe，点击 root 按钮，若期间弹出对话框，确认即可。提权后安装 ROOT Explorer
  • 删除定制应用，位置：/cust/chinatelcom/cn/app & /system/app

C8650

目前无法使用 z4root 以及 SuperOneClick，必须在刷过 Recovery 后刷入 ROOT 包。

  1. 刷 Recovery 包
    • 在手机端：设置 - 应用程序 - 开发 - 勾选 USB 调试
    • 在 PC 上安装手机驱动程序
    • 关机，按住 音量上键 + 电源键 10秒以上进入刷机模式
    • 用数据线连接 PC，开始刷 Recovery
  2. 刷 ROOT 包
    • 拷贝 root.zip 至 SD 卡，关机
    • 按住 音量上键 + 电源键进入 Recovery 模式
    • 选择 install zip from sdcard，选择 root.zip 包 ，完成
    • 之后可以使用 ROOT Explorer 删除定制应用。位置：/cust/chinatelcom/cn/app & /data/datapp & /system/app
