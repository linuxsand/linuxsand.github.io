使用 http://stackoverflow.com/a/5606276 提及的方法（External Tool），可以在 VS 中执行自定义脚本。

假设，在某些时刻，我想把构建完成的可执行文件拷贝到特定路径，且不手工执行脚本（还不是懒么）。

那么可以这样：

  • 编写拷贝脚本（不提），储存于 C:\copyto.bat
  • 打开 VS，打开 Tools 菜单内的 External Tools，填写如下：
  • 再打开 Tools 菜单内的 Customize 项
    • 选择 Commands 标签页，我希望在工具栏中使用它，选工具栏
    • 添加命令，在弹出的窗口中，选择 Tool 分类的“外部命令#”（这是我的第一个命令）
  • 关闭 Customize 窗口

接下来，就可以点击工具栏的按钮去执行脚本了。
