FANUC 控制器内建了一个 web server，它暴露了部分控制器的内部细节，不仅限于标题所说的「快速添加 IO 注释」。

1 如何在电脑上访问机器人控制器的 web server

需要做一些设置，分为两方面，一是机器人控制器端的配置，二是电脑端的配置。

机器人控制器端

  • 先配置机器人控制器 IP：Menu - Setup - Host Comm - TCP/IP，给 port 1 分配一个 IP（我这里是 192.168.20.201，下面以此为例）
  • 再设置 HTTP 服务的用户名、口令：Menu - Setup - Host Comm - HTTP，设置第二行 KAREL:* 的 Name / Pwd 字段（当然，你也可以不配置，只需 unlock 这一行，默认是 auth，即需要校验口令）
  • 将网线的一头插在 port 1，注意是 port 1

电脑端

只需设置一个同网段的 IP 即可，我这里设置了 192.168.20.202。插上网线。

用浏览器打开 http://192.168.20.201（机器人 IP），在弹出的对话框里输入用户名、口令，就可以看到如下页面。

这里可以看到机器人内部的很多东西，善加利用可以提高调试效率。

2 如何快速添加注释：官方的网页版工具

页面最底部有一个 Comment Tool，点击即可进入。（http://192.168.20.201/KAREL/COMMAIN）

点击右侧的 Digital I/O，即可给数字输入输出添加注释。一个很贴心的地方是，当你在文本框中输入注释、光标移动到下一个文本框时（unfocus），这个注释就被储存到机器人控制器内部了。
